# AGENTS — BAH 2026, Challenge 15

## Bharatiya Antariksh Hackathon 2026

**Organizer:** ISRO × Hack2skill  
**Team:** IISER Kolkata  
**Challenge:** #15 — Forecasting and/or Nowcasting of Solar Flares using combined Soft and Hard X-ray data from Aditya-L1

---

## Problem Statement

Solar flares are sudden, intense bursts of radiation originating from the release of magnetic energy in the solar atmosphere. These events can trigger severe space weather disruptions, impacting Earth-bound satellite communications, GPS navigation, and power grids. Anticipating these disruptions is critical for infrastructure safeguarding.

ISRO's Aditya-L1 mission continuously monitors the Sun from Lagrange Point L1 using two X-ray payloads:
- **SoLEXS** (Solar Low Energy X-ray Spectrometer) — captures soft X-rays
- **HEL1OS** (High Energy L1 Orbiting X-ray Spectrometer) — captures hard X-rays

This challenge requires participants to build an automated algorithmic pipeline that uses the combined time-series data from SoLEXS and HEL1OS to detect (nowcast) or predict (forecast) solar flares.

---

## Objectives

1. **Nowcasting:** Design an automated solar flare detection algorithm for real-time detection and classification of solar flares using soft and hard X-ray data.

2. **Forecasting:** Build a predictive algorithm that identifies precursor patterns in soft and hard X-ray light curves before a solar flare occurs.

---

## Expected Outcomes

- An automated database of nowcasted solar flares, generated by combining soft and hard X-ray light curves
- A trained model capable of forecasting flares with a quantifiable lead time
- An interface that visualizes the X-ray light curves and triggers visual alerts when a flare is nowcasted or forecasted

---

## Expected Solution Steps

### 1. Data Acquisition
- Download SoLEXS and HEL1OS Level-1 data from the ISSDC PRADAN portal
- Use the provided shell scripts in `data/downloads/`

### 2. Data Reading & Exploration
- Write scripts to read SoLEXS and HEL1OS time-series light curve data
- Visually understand typical solar flare light curves observed in soft and hard X-rays

### 3. Algorithmic Nowcasting
- Build algorithms to detect flares independently in both soft and hard X-rays
- Combine these independent catalogues to generate a master catalogue

### 4. Predictive Modelling
- Train a time-series model using the combined dataset
- Predict the probability of a flare occurring in the next N minutes

### 5. Visualization & Alerting
- Build a dashboard showing X-ray light curves
- Trigger visual alerts on nowcasted/forecasted flares

---

## Evaluation Criteria

- Detection of low- and high-class flares
- High True Positive Rate and low False Alarm Rate for predictions
- Lead Time of predictions: how many minutes before the flare peak the model triggers an alert
